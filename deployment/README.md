# Deployment

Die Intune-Skripte liegen in diesem Repository unter:

```text
deployment/intune/
```

Die veröffentlichte EXE muss heißen:

```text
GSL-Service-Center.exe
```

Intune Detection per Dateiversion:

```text
Pfad: C:\GSL-Support\GSL-Service-Center
Datei: GSL-Service-Center.exe
Methode: String (version)
Operator: Greater than or equal
Wert: 0.5.13.0
```
